A premium, production-grade memorial preservation platform. Create lasting digital memorials, share tributes, and discover resting places through an interactive map.

**Status: Production-Ready**

## Features
- **Digital Memorials**: Create beautiful memorial pages with photos, life stories, and tributes
- **Interactive Map**: Plot and discover memorials worldwide using Leaflet
- **Service Booking**: Schedule memorial services, visitations, and events
- **Photo Gallery**: Upload and organize photo collections
- **Guest Book**: Leave messages, share memories, and connect with others
- **Insights Dashboard**: Track visits, tributes, and engagement analytics
- **Authentication**: User accounts with sign in/register functionality
- **PWA Support**: Manifest configured for progressive web app capabilities

## How to Run
Open `index.html` in any modern web browser. No server required.

```bash
# Clone the repo
git clone https://github.com/CodeBreweryZA/Umzukulu-.git

# Open directly in browser
cd Umzukulu-
start index.html
```

## Tech Stack
- HTML5
- CSS3 with custom properties (forest-green/gold theme)
- JavaScript (Vanilla ES6+)
- Leaflet.js for interactive maps
- Font Awesome 6 for icons
- Google Fonts (Inter, Playfair Display)
- LocalStorage for data persistence
- PWA Manifest
